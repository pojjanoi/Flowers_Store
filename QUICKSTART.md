# 🚀 Quick Start Guide - เริ่มต้นใช้งานแบบรวดเร็ว

## ⚡ เริ่มต้นใช้งาน (3 ขั้นตอนเบื้องต้น)

### ขั้นตอนที่ 1️⃣ : ติดตั้ง Flask

```bash
pip install -r requirements.txt
```

### ขั้นตอนที่ 2️⃣ : รัน Flask Application

#### 🪟 สำหรับ Windows:
```bash
python app.py
```

หรือคลิก `run.bat` ไฟล์เพื่อรันอัตโนมัติ

#### 🐧 สำหรับ Linux/Mac:
```bash
python3 app.py
```

หรือรัน `run.sh`:
```bash
chmod +x run.sh
./run.sh
```

### ขั้นตอนที่ 3️⃣ : เปิดเบราว์เซอร์

```
http://localhost:5000
```

## 📖 วิธีใช้งานระบบ

### 🏠 หน้าแรก (Home Page)
- แสดงรายการดอกไม้ทั้งหมดในรูปแบบ Card
- ดูราคา จำนวน และสถานะสต็อกของแต่ละรายการ
- ปุ่มดำเนินการ: ดู (👁), แก้ไข (✏️), ลบ (🗑️)

### ➕ เพิ่มดอกไม้ใหม่
1. คลิกปุ่ม "🌸 เพิ่มดอกไม้ใหม่"
2. กรอกข้อมูล:
   - **ชื่อดอกไม้**: เช่น "กุหลาบแดง"
   - **หมวดหมู่**: เลือกจากรายการ
   - **ราคา**: เป็นบาท (ต้องมากกว่า 0)
   - **จำนวน**: จำนวนชอ (≥ 0)
   - **รายละเอียด**: (ตัวเลือก)
3. คลิก "✅ เพิ่มดอกไม้"

### 👁️ ดูรายละเอียด
- คลิกปุ่ม "ดู" บนการ์ดดอกไม้
- จะแสดงข้อมูลทั้งหมดอย่างละเอียด
- สามารถแก้ไข ลบ หรือกลับไปหน้ารายการได้

### ✏️ แก้ไขข้อมูล
1. คลิกปุ่ม "แก้ไข" บนการ์ดดอกไม้
2. แก้ไขข้อมูลที่ต้องการ
3. คลิก "💾 บันทึกการแก้ไข"

### 🗑️ ลบรายการ
1. คลิกปุ่ม "ลบ" บนการ์ดดอกไม้ หรือหน้ารายละเอียด
2. ยืนยันการลบในป็อปอัป
3. รายการจะถูกลบออกจากระบบ

## 🐛 การแก้ไขปัญหาทั่วไป

### ❌ ข้อผิดพลาด: "No module named 'flask'"
**วิธีแก้:**
```bash
pip install Flask==2.3.3
```

### ❌ ข้อผิดพลาด: "Port 5000 is already in use"
**วิธีแก้:** ปิดแอปพลิเคชันอื่นที่ใช้ Port 5000 หรือเปลี่ยน Port ใน `app.py`:
```python
app.run(debug=True, host='0.0.0.0', port=5001)  # เปลี่ยนจาก 5000 เป็น 5001
```

### ❌ ข้อผิดพลาด: "database is locked"
**วิธีแก้:** ปิด Flask แล้วรันใหม่ หรือลบไฟล์ `flowers_store.db` และรัน `create_database.py` ใหม่

### ❌ ข้อผิดพลาด: "no such table: Flowers"
**วิธีแก้:** รัน script สร้างฐานข้อมูล:
```bash
python create_database.py
```
